# 中华美食网站 | Chinese Cuisine Website

一个连接中西饮食文化的双语网站，帮助海外用户学习中餐烹饪。

## 项目特色

- **中英双语对照** - 支持中文/英文一键切换
- **浅色主题设计** - 清爽现代的视觉风格
- **场景化分类** - 按实际使用场景分类菜谱（留学生一人食、Potluck聚会、节日庆典、初学者友好）
- **量化步骤** - 精确的时间和用量说明，解决"适量""火候"等抽象概念
- **食材替代方案** - 针对海外食材获取情况提供替代建议
- **文化背景故事** - 每道菜都配有文化背景介绍，传播中华饮食文化

## 技术栈

- 纯 HTML/CSS/JavaScript
- 响应式设计，支持移动端
- 无需构建工具，直接在浏览器中运行

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 点击右上角的语言按钮切换中英文
3. 通过导航栏浏览不同页面
4. 点击菜谱卡片查看详细制作步骤

## 主要功能

### 首页
- 展示网站主题和愿景
- 场景化菜谱分类入口
- 精选菜谱展示

### 场景分类页面
- 留学生的一人食慰藉
- 西方人的Potluck聚会惊艳单品
- 节日庆典必备
- 初学者友好

### 菜谱详情页面
- 食材清单（含替代方案）
- 量化制作步骤（含时间）
- 文化背景故事

### 文化故事页面
- 中式烹饪哲学
- 锅气概念
- 调味平衡
- 餐桌礼仪

## 示例菜谱

目前包含4道经典中餐菜谱：

1. **西红柿炒鸡蛋** - 初学者友好，15分钟
2. **宫保鸡丁** - 中级难度，25分钟，适合Potluck
3. **红烧肉** - 高级难度，90分钟，节日庆典
4. **蒜蓉西兰花** - 初学者友好，10分钟
