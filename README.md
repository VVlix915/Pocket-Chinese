# 中华美食网站 | Chinese Cuisine Website

一个连接中西饮食文化的双语网站，帮助海外用户学习中餐烹饪。

## 项目特色

- **中英双语对照** - 支持中文/英文一键切换
- **浅色主题设计** - 清爽现代的视觉风格
- **场景化分类** - 按实际使用场景分类菜谱（留学生一人食、Potluck聚会、节日庆典、初学者友好）
- **量化步骤** - 精确的时间和用量说明，解决"适量""火候"等抽象概念
- **食材替代方案** - 针对海外食材获取情况提供替代建议
- **文化背景故事** - 每道菜都配有文化背景介绍，传播中华饮食文化

## 技术栈

- 纯 HTML/CSS/JavaScript
- 响应式设计，支持移动端
- 无需构建工具，直接在浏览器中运行

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 点击右上角的语言按钮切换中英文
3. 通过导航栏浏览不同页面
4. 点击菜谱卡片查看详细制作步骤

### 模块说明

#### JavaScript模块
- **app.js**: 应用入口，负责初始化和启动应用
- **state.js**: 管理应用状态（当前语言、页面、选中的菜谱等）
- **utils.js**: 提供工具函数，如翻译功能
- **router.js**: 处理页面导航和语言切换
- **render.js**: 包含所有页面渲染逻辑

#### 数据文件
- **translations.js**: 中英文翻译对照表
- **recipes.js**: 所有菜谱的详细信息（食材、步骤、文化背景等）

#### 静态资源
- **assets/images/covers/**: 菜谱封面图片
- **assets/images/cultural/**: 文化背景说明图片
- **assets/videos/**: 烹饪视频教程

## 扩展计划

### 架构优势（拓展性提升）

新的模块化架构提供了更好的可扩展性：

- **数据与逻辑分离**: 菜谱数据和翻译数据独立存储，便于添加新菜谱或支持新语言
- **模块化设计**: 各功能模块独立，修改某个功能不会影响其他部分
- **易于维护**: 代码结构清晰，便于定位和修改特定功能
- **可扩展性强**: 
  - 添加新语言：只需在 `translations.js` 中添加新的语言对象
  - 添加新菜谱：在 `recipes.js` 中添加新的菜谱对象即可
  - 添加新功能：可以轻松添加新的模块而不影响现有代码
  - 自定义渲染：渲染逻辑独立，可以轻松定制页面样式
